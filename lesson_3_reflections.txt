When would you want to use a remote repository rather than keeping all your work
local?

    Remote repository helps keep the repository online as keeping all your work local might be dangerous as we risk losing data in case of unforeseen incidents.

automatically stay up-to-date with your remote repository?

    Github cannot update the changes that I have made to the files locally. So, to update the repository with all the changes you have, you must pull the repository, add the changes, commit them and then puh them back.
